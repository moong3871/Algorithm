# 탐욕 알고리즘

> 여러 경우 중 하나를 결정해야 할 때 마다, 그 순간에 최적이라고 생각되는 것을 선택

1. 해 선택

   부분 문제의 최적 해 -> 부분 해 집합에 추가

2. 실행 가능성 검사

   새로운 부분 해 집합이 실행 가능한지 확인

   (문제의 `제약 조건 위반 여부` 검사)

3. 해 검사

   새로운 부분 해 집합이 `문제의 해가 되는지` 확인
