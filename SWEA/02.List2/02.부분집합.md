## 비트 연산자

> 0과 1로 이루어진 이진수에 대한 연산을 수행하는 연산자

```
비트 연산자 종류
& : 비트 단위로 AND 연산, 둘다 1이면 1 아니면 0
| : 비트 단위로 OR  연산, 둘다 0이면 0 아니면 1
value << 개수: value의 비트열을 왼쪽으로 개수만큼 이동
>> : 피연산자의 비트열을 오른쪽으로 이동
```

```
3 << 2 -> 0011 << 2 -> 1100 -> 12
3 >> 2 -> 0011 >> 2 -> 0000 -> 0
```
